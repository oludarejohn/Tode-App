# Project Specification: TaskFlow

## Project Overview
TaskFlow is a modern, user-friendly Todo application that helps users manage their daily tasks efficiently.

## Functional Requirements
- Users can add new tasks
- Users can mark tasks as completed
- Users can delete individual tasks
- Tasks persist after page refresh using LocalStorage
- Clean statistics showing total and completed tasks

## Technical Requirements
- Pure HTML, CSS, and JavaScript (No frameworks)
- Responsive design
- Modern, clean UI with smooth interactions
- Deployable on GitHub Pages

## Future Enhancements
- Due dates
- Categories / Priority levels
- Dark mode toggle
- Task filtering (All/Active/Completed)
